About the Project

Background
This Credit Card Approval Prediction web application is a capstone project developed for COMP 7980. It aims to apply the following:

  • Concepts learned from my computer science and statistics classes as a data engineering student
  • Personal interest in web development and web accessibility
  • Career experience in DevOps

Objective
This project aims to simulate a credit card approval system. A machine learning model has been trained on an existing dataset, and the trained model has been serialized to serve as the web application's backend. Based on the user's inputs with the form on the frontend, the model will then determine whether the user is likely to be approved for a credit card.

Dataset and Model
The dataset used to train the machine learning model can be found here. More information about how the dataset was obtained can be found on the Dataset page and how the dataset was engineered for model training can be found on the EDA page. The process of training the machine learning model is documented on the ML Model page.
