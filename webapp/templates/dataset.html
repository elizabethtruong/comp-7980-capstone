Dataset Information

Origin
The dataset used to train the machine learning model can be found here. This particular dataset is a cleaned, merged, and transformed version of this dataset. The data is sourced from an unknown Chinese financial institution which led to some differences I had to account for during feature engineering, such as Yuan to USD conversion for total income.

Contents
The dataset contains 25,128 observations, 20 features, and 1 target in total. The target variable is Status and represents whether the applicant is approved or denied for credit. After data preprocessing and feature engineering detailed on the EDA page, the number of features were reduced to 10 which are the ones present on the prediction form. By reducing the number of features, the model can produce accurate and reliable results from the features determined to be meaningful.

Reflection
A downside to using this dataset was that it did not provide data for credit score or total assets. Such information would have been valuable in model training and would have helped in more closely resembling a credit card application in the real world. However, I could not find an adequate dataset with these features. Otherwise, I thought this dataset provided a good challenge during data evaluation and model training.
